📄 Sito Statico

+

Un sito composto da file "finiti" (HTML, CSS, JavaScript). Il server lo consegna al browser dell'utente esattamente come lo hai scritto, senza elaborazioni complesse (niente database o PHP).

🔄 Git

+

È un sistema di "controllo di versione". Immaginalo come una macchina del tempo per il tuo codice: registra ogni modifica e ti permette di tornare indietro se qualcosa si rompe.

🗄️ Repository (Repo)

+

È la cartella remota su GitHub che ospita il tuo progetto online e ne conserva tutta la cronologia.

⚙️ GitHub Actions

+

È un sistema di automazione. Legge le tue modifiche e "costruisce" il sito live ogni volta che carichi un aggiornamento.
